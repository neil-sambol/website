TMX Finance

Project Details

  • Category : Reports
  • Domain : B2C Lending

BUSINESS PROBLEM:

How to keep the stakeholders in the loop for a complicated internal database upgrade in a fashion they will understand.


SOLUTION:

I have elided the textual commentary and left the charts to speak for themselves. In the first chart, one can see the clear burndown of the test cases remaining (red) and of those that have failed (purple). It can also be seen that the test cases passed (green) are trending up.

Two additional benefits of this report are that one can see that very little additional scope is being added to the project and also that the number of test cases blocked is increasing (not good but provides information as to which part of the project to focus upon).


EFFECTIVENESS:

This simple report is able to show in near-real-time, the status of a project. I find that frequent updates of information in a form easily digestable to stakeholders increases transparency and trust within any project.
